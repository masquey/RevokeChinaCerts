## 使用方法 ##
SoftCertPolicyAppender.exe <参数> <证书路径>

SoftCertPolicyAppender.exe <参数> <证书路径> <证书路径2> ...

**参数说明**

- --set-force 启用强制策略
- --unset-force 取消强制策略
- -r 移除证书规则
- -h 显示帮助信息


## 备注 ##
- 需要管理员权限 
- 需要[.net framework 4.0](http://www.microsoft.com/en-us/download/details.aspx?id=17718)
